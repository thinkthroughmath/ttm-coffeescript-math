    Precision

  • ¶

    A library for doing ‘precise’ (i.e. not-hindered-by-floating-point-garbage) math. It is slow, hacky, and awful, but it does work for our purposes.

    See the precise library and the precise library specs

        precise = opts.precise || ttm.lib.math.Precise.build()

    ‘Components’

  • ¶

    represent the various components of an expression. (e.g. ‘numbers’, ‘variables’, ‘exponentiations’)

    See the components library and the components library specs

    @equation = opts.equation || comps.equation
    @expression = opts.expression || comps.expression

  • ¶
